# SALES-DASHBOARD
Developed an interactive Power BI Sales Analytics Dashboard to clean, model, and visualize sales data, enabling trend analysis and data-driven business insights.

Objective
To design a user-friendly dashboard that converts raw sales data into actionable business insights for performance tracking and strategic planning.

Project Summary
This project is an interactive Sales Analytics Dashboard built using Power BI to analyze sales performance across countries, products, and time periods. The dashboard transforms raw sales data into meaningful insights through data cleaning, modeling, and visualization. It helps identify trends, top-performing products, and monthly growth patterns using dynamic charts and KPIs. The project demonstrates practical business intelligence skills and the ability to convert complex datasets into clear visual stories for decision-making.

Tech Stack
Power BI
Power Query
DAX (Data Analysis Expressions)
Microsoft Excel
Data Modeling
Data Visualization

Tools Used
Power BI Desktop
